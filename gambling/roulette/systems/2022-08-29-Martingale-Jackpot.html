Here's a much simpler betting pattern with similar results that is a lot easier to bet:

  • You still push on 21 numbers.
  • You still lose {% include win-loss-amount.html amount=-12 %} on 9 numbers.
  • You still win {% include win-loss-amount.html amount=6 %} on 6 numbers.
  • You still win {% include win-loss-amount.html amount=24 %} on 2 numbers.

This betting pattern is a lot easier to make AND it will be easier to remember which numbers win, which ones lose, and which ones push.
